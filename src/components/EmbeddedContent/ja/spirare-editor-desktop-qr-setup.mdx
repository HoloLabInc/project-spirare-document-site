ダウンロードした glb モデルをアプリウィンドウにドラッグ&ドロップし、シーンに追加してください。  
さらに、右側の **INSPECTOR** パネルにて以下のように設定してください。

```
Clickable in Editor: 無効
AR Display Mode: hidden
Space origin: 有効
Space origin space id: <QR コードに格納された文字列>
Scaling: <QR コードのおおよそのサイズを設定>
```
