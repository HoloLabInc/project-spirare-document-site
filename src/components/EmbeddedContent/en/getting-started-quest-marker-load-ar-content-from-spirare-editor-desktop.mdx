Press the menu button on the left controller to open the app menu.
In the POML URL field, enter the URL displayed by the Spirare Editor Desktop (`http://<PC's IP Address>:8080`).
Entering `http://` is optional.

Press **Load** to load the content; a **Loaded successfully** message will appear once loading finishes.
If you make changes to the content in Spirare Editor after it has been loaded, the display in Spirare Browser will be automatically updated.
