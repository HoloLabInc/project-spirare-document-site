This guide explains how to create and display AR content using Project Spirare.

In this guide, iOS or Android devices are used as display devices, and the QR Code marker is utilized for alignment.

## Required Devices

<GettingStartedRequiredDevices />

Additionally, the AR device and PC must be connected to the same network, and communication between them must be permitted.

## Preparation

### Install Spirare Browser

Please install the Spirare Browser app for QR marker alignment (display name `SB Marker`) on the AR device using one of the following methods.

#### Install Prebuilt Application (iOS)

<SpirareBrowserAppInstallIos />

#### Install Prebuilt Application (Android)

<SpirareBrowserAppInstallAndroid />

#### Install from Source

If you want to install from source, please refer to [Install from Source](/spirare-browser/spirare-browser-arfoundation-marker.mdx#install-from-source).

### Download Spirare Editor Desktop

<SpirareEditorInstall />

### Prepare the QR Code

Please create a QR code that contains an alphanumeric string.
The stored string will be used as an ID for alignment.

In this guide, a printed PDF from the link below is used.
The string contained in the QR code is `QR`.

Detecting a QR code and loading the AR content will display the AR content.
You can perform these two steps in any order.
