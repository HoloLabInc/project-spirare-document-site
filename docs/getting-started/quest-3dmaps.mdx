This guide explains how to create and display 3D content using Project Spirare.

In this guide, Meta Quest is used as display devices, and 3D content is shown on a 3D Map.

## Required Devices

<GettingStartedRequiredDevices />

Additionally, the Meta Quest and PC must be connected to the same network, and communication between them must be permitted.

## Preparation

### Install Spirare Browser

You will need to build and install the 3D map version of the Spirare Browser app (display name `Spirare Browser 3DMaps`).
Please refer to the [Install from Source](/spirare-browser/spirare-browser-quest-3dmaps.mdx#install-from-source) for installation instructions.

## Display 3D Content

### Check the Meta Quest IP Address

In the Settings menu, select Wi-Fi, then choose the network you are currently connected to.
Check the IP address which is displayed in the details of the connected network.

### Launch the Spirare Browser App

Please launch the 3D map version of the Spirare Browser app (display name `Spirare Browser 3DMaps`) on the Meta Quest.

By operating the side of the map base with the controller, you can move, rotate, and scale the map object.  
By operating the top of the map base with the controller, you can move and zoom the displayed area of the map.

<img width="640" class="border" src={SpirareBrowserControllersImage} />

If the placed content is outside the visible area, move the map or zoom out to adjust the display range.
