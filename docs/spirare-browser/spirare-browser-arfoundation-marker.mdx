# Spirare Browser Marker (SB Marker)

Spirare Browser Marker (display name SB Marker) is the iOS/Android version of the Spirare Browser app that uses the QR Codes for alignment.

## Install Prebuilt Application

### For iOS Devices

<SpirareBrowserAppInstallIos />

### For Android Devices

<SpirareBrowserAppInstallAndroid />

## Install from Source

### Clone the Repository

<SpirareBrowserCloneRepository />

### Open the Unity Project

Please open the `unity/SpirareBrowser-ARFoundation-Marker` folder in the Unity Editor.
Please use Unity version 2022.3.16f1.

### Build the Project

Change the platform to Android or iOS, and build and deploy the application to the device.

<img width="480" src={BuildSettings} />

## Usage

### Detect QR Codes

<SpirareBrowserQRDetection />

### Load POML from a Network

Enter the URL that delivers the POML in the side menu, and press the **Load** button.
Entering `http://` is optional.

<img width="480" class="border" src={SpirareBrowserContentImage} />

### Load POML from a Local File

<SpirareBrowserARFoundationMarkerPomlPlacement />

The `.poml.zip` files placed in the device will be automatically loaded when the app is launched.
